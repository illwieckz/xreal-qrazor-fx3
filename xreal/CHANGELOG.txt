__________________________________________

	* new XreaLRadiant level editor based on DarkRadiant
	* redesigned new menu
	* redesigned new hud
	* replaced almost all font rendering with True Type fonts
	* added Prey style wallwalking
	* changed media license for most media files to http://creativecommons.org/licenses/by-nc-sa/3.0/
	* new sounds
	* new mapobject models
	* new MD5 player model
	* new textures
	* new maps
	* new weapons with alternate firemodes
	* new true HDR rendering with adapative tone mapping that includes HDR lightmapping
	* new non-uniform lighting that allows mixing of static precomputed lighting and realtime lighting
	* improved bloom post process effect
	* improved texture compression
	* improved OpenGL interface designed for the upcoming OpenGL 3.0 forward compatible context
	* improved ATI support
	* improved hardware accelerated BSP/PVS rendering
	* new GPU accelerated MD5 skeletal animation system that outperforms all current CPU based implementations
	* core engine based on the latest ioquake3 1.36 code
	* new Blender 3D exporter scripts
	* tons of bugfixes and overall performance optimizations
